# Mon site GitHub Pages

## Démarrage rapide
1. Renommez `SITE.name`, `SITE.githubUser`, `SITE.email` dans `assets/js/site.js`.
2. Remplacez `assets/img/photo.jpg` par votre portrait (carré) et `assets/cv.pdf` par votre vrai CV.
3. Créez un dépôt **username.github.io** (ou activez Pages sur n'importe quel dépôt).
4. Uploadez tout le contenu de ce dossier à la racine de la branche par défaut.
5. Ouvrez `https://username.github.io` (ou l'URL Pages du dépôt).

## Pages
- `index.html` — About (accueil)
- `cv.html` — CV
- `code.html` — Repos GitHub (auto)
- `research.html` — Research interests
- `publications.html` — Publications
- `blog/` — Blog
- `contact.html` — Contact (mailto)

Bonne personnalisation !
